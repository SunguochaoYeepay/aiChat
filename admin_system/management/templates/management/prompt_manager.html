提示词模板使用说明

提示词模板概念

提示词模板是预定义的文本模式，用于指导AI模型的行为和回答方式。通过定制不同场景的提示词模板，可以使AI输出更符合特定需求的内容。

模板类别

  • chat - 文本聊天类提示词，用于一般对话场景
  • image_analysis - 图像分析类提示词，用于分析图片内容
  • search - 知识库搜索类提示词，用于基于知识检索的回答
  • topic_matching - 主题匹配类提示词，用于识别查询的主题类别

模板变量

提示词模板中可以使用特定变量，系统会在处理时自动替换：

  • {query} - 用户的查询内容
  • {history} - 聊天历史记录
  • {content} - 从知识库检索到的内容

开发者接口

在代码中调用提示词模板：

from management.prompt_manager import prompt_manager

# 获取聊天提示词
prompt = prompt_manager.apply_chat_template(query, history, template_type='expert')

# 获取图像分析提示词
prompt = prompt_manager.apply_image_analysis_template(query, template_type='detail')

# 获取搜索提示词
prompt = prompt_manager.apply_search_template(query, content, template_type='rag')

API参数

调用API时可以通过template_type参数指定使用的模板类型：

# 聊天API示例
POST /api/v1/chat/completions
{
    "messages": [...],
    "template_type": "expert"
}

# 图像分析API示例
POST /api/analyze
{
    "image_base64": "...",
    "query": "分析这张图片",
    "template_type": "detail"
}
