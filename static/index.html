本地 Qwen-VL-Chat API 服务

服务说明

本服务提供基于通义千问-VL-Chat的多模态模型API，支持文本聊天和图片分析功能。

服务基于FastAPI开发，已启用CORS，可以从任意前端应用调用。

API列表

  • POST /v1/chat/completions - 文本聊天API，兼容OpenAI格式
  • POST /analyze - 图片分析API，支持上传图片并提问
  • POST /search - 知识库检索API，支持按话题检索知识
  • POST /refresh_kb - 刷新知识库
  • GET /topics - 获取所有知识库话题
